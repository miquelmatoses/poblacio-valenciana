## Design System (mm-design)

This README follows mm-design visual conventions (https://github.com/miquelmatoses/mm-design).

### Rules
1. **Badge colors**: All shields.io badge hex colors must use the mm-design brand palette:
   - Red: cf3339 | Blue: 0047ba | Yellow: f1c22f | Green: 427c42 | Black: 111111
2. **Fonts**: Roboto in typing-svg and any visual elements. No Fira Code.
3. **No arbitrary hex colors**: Always use the closest mm-design brand color.
